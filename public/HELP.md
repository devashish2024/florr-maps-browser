# Help

Need help using FlorrMapBrowser? Here's a quick guide.

---

## Opening Maps

1. In the **Explorer** panel on the left, expand the **maps** folder. <br/>
2. Click any map name (e.g. Garden, Desert, Ocean) to open it. <br/>
3. The map renders in the main area — drag to pan, scroll to zoom. <br/>
4. **Middle-click** (mouse wheel click) to reset zoom to default.

## Viewing Tiles

1. Expand the **tiles** folder in the Explorer. <br/>
2. Click any tile SVG to see it rendered at full size.

## Viewing File Contents

- Click **tileset.tsj** to see the raw tileset definition. <br/>
- Click **maps.txt** to see the raw map list from florr.io.

## Map Interactions

- **Hover** over colored zones to see spawner info — difficulty, density, and which mobs spawn there. <br/>
- **Checkpoints** are shown as magenta outlines. Hover to see level requirements. <br/>
- **Mob sprites** appear in tooltips when hovering spawner zones.

## Sidebar

- The sidebar is **resizable** — drag the edge to adjust width. <br/>
- On mobile, tap the **☰** button to open the sidebar. <br/>
- Maps that failed to load appear at the bottom with a ⚠️ icon.

## Context Menu

- **Right-click** any file in the Explorer to open a context menu. <br/>
- **Copy URL** — copies the original florr.io URL for that file. <br/>
- **Open URL** — opens the original URL in a new browser tab. <br/>
- README.md and HELP.md are local files and don't have remote URLs.

## Keyboard & Mouse

| Action | Control |
| Pan | Click + drag |
| Zoom in/out | Scroll wheel |
| Reset zoom | Middle-click |
| Pinch zoom | Two-finger pinch (touch) |
